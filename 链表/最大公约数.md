`gcd` 是 **最大公约数**（Greatest Common Divisor）的缩写。

## 什么是最大公约数？
最大公约数是指两个或多个整数共有约数中最大的一个。

## 举例：
- `gcd(8, 12) = 4`（因为8和12都能被4整除）
- `gcd(15, 25) = 5`
- `gcd(7, 13) = 1`（两个质数，最大公约数为1）
- `gcd(0, 5) = 5`

## 在C++中的使用：

### 1. C++17 标准库（推荐）
```cpp
#include <numeric>
int result = std::gcd(a, b);
```

### 2. 自己实现（辗转相除法）
```cpp
int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

# **辗转相除法**（欧几里得算法）是求两个数最大公约数的高效方法。

## 算法原理
基于定理：**gcd(a, b) = gcd(b, a mod b)**

## 计算过程
用较大数除以较小数，再用余数替换较大数，重复直到余数为0。

## 举例：gcd(48, 18)
```
48 ÷ 18 = 2 余 12
18 ÷ 12 = 1 余 6  
12 ÷ 6  = 2 余 0   ← 余数为0，结束
∴ gcd(48, 18) = 6
```

## 代码实现

### 1. 循环版本
```cpp
int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

### 2. 递归版本
```cpp
int gcd(int a, int b) {
    if (b == 0) return a;
    return gcd(b, a % b);
}
```

## 逐步演示 gcd(48, 18)

| 步骤 | a  | b  | a % b | 说明 |
|------|----|----|-------|------|
| 1 | 48 | 18 | 12 | 48 ÷ 18 = 2 余 12 |
| 2 | 18 | 12 | 6  | 18 ÷ 12 = 1 余 6 |
| 3 | 12 | 6  | 0  | 12 ÷ 6 = 2 余 0 |
| 4 | 6  | 0  | -  | b=0，返回a=6 |
